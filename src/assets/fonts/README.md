# Font files

Add **Schnyder** here for the design system heading font.

- **Filename:** `Schnyder-Regular.woff2` (or update the path in `src/design-system/fonts.ts`)
- Schnyder is a commercial typeface from [Commercial Type](https://commercialtype.com/catalog/schnyder). You need a license to use it.
- After adding the file, replace the `fontHeading` export in `src/design-system/fonts.ts` with the `localFont` version (see comment at top of that file).
